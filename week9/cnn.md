# CNN(합성곱 신경망) 알고리즘 핵심 요약

CNN(Convolutional Neural Network)은 이미지의 공간적 정보를 유지하면서 특징을 효과적으로 학습하는 딥러닝 구조입니다.

---

## 1. CNN의 정의와 필요성
컴퓨터가 시각적 정보를 분석할 때, 이미지를 1차원으로 늘어뜨리지 않고 **격자 형태(공간 구조)** 그대로 처리하여 사물의 특징을 찾아내는 신경망입니다.

* **Dense(전결합) 모델의 한계:** 이미지가 조금만 회전하거나 이동해도 다른 데이터로 인식함.
* **CNN의 해결책:** 필터를 통해 이미지의 패턴(선, 모양 등)을 직접 찾아내어 위치 변화에 강함.

## 2. 주요 구성 요소

### ① 합성곱 층 (Convolutional Layer)
* **작동 방식:** 필터(Filter)가 이미지 위를 훑으며 `y = wx + b` 연산을 수행합니다.
* **특징 추출:** 이미지에서 경계선, 질감 등 핵심적인 특징 지도를 생성합니다.
* **가중치 공유:** 똑같은 필터를 이미지 전체에 사용하므로 학습해야 할 파라미터가 적어 매우 효율적입니다.

### ② 풀링 층 (Pooling Layer)
* **작동 방식:** 데이터의 크기를 줄이는 **압축** 단계입니다.
* **Max Pooling:** 정해진 구역에서 가장 큰 값만 남기고 나머지는 버립니다.
* **장점:** 계산량이 줄어들고, 사물의 위치가 미세하게 변해도 동일한 특징으로 인식하게 합니다.



### ③ 전결합 층 (Fully Connected Layer)
* **작동 방식:** 추출된 2차원 특징들을 1차원으로 길게 펼치는 **Flatten** 과정을 거칩니다.
* **최종 분류:** 앞서 모은 특징들을 종합하여 "고양이일 확률 90%"와 같은 최종 결론을 내립니다.

## 3. Dense 층 vs CNN 비교

| 구분 | Dense (전결합) | CNN (합성곱) |
| :--- | :--- | :--- |
| **연결 방식** | 모든 입력-출력 노드가 연결됨 | 필터 범위 내의 노드들만 연결됨 |
| **공간 정보** | 파괴됨 (1차원 변환 필수) | 보존됨 (2차원/3차원 유지) |
| **효율성** | 입력이 클수록 계산량 폭증 | 필터 공유로 매우 경제적임 |

---

## 4. 요약된 처리 흐름
1. **Input:** 원본 이미지 데이터
2. **Feature Extraction:** Convolution + Pooling 반복 (특징 찾기)
3. **Classification:** Flatten + Fully Connected (정답 예측)
